🔢 Design de Paginação – Navegação Numérica para Sites

Este projeto apresenta uma barra de paginação elegante e responsiva, desenvolvida com HTML e CSS. Ideal para sites com múltiplas páginas de conteúdo, como blogs, catálogos, fóruns ou sistemas administrativos, oferecendo uma navegação clara e intuitiva.

📄 Descrição

A interface inclui:

- Botões de navegação “Anterior” e “Próximo”
- Numeração de páginas com destaque para a página ativa
- Elemento de separação (“...”) para indicar páginas intermediárias
- Ícones de seta via Font Awesome
- Layout centralizado com estilo moderno
  
A estrutura é simples e eficaz, permitindo fácil adaptação para diferentes contextos e sistemas. A paginação é visualmente agradável e melhora a experiência do usuário em sites com grande volume de conteúdo.

🎨 Estilos CSS

O arquivo style.css contém:

- Reset básico: remoção de margens e preenchimentos padrão
- Estilo do Body:
- Fundo cinza claro
- Centralização com Flexbox
- Container de paginação:
- Fundo branco com bordas arredondadas
- Sombra suave para profundidade
- Lista de páginas:
- Layout horizontal com espaçamento uniforme
- Links com transição de cor e fundo
- Destaque visual para página ativa
- Botões de navegação:
- Estilo diferenciado com ícones
- Efeito hover com mudança de cor
